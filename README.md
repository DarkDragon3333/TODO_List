# Терминальное приложение "Список задач"

---

## Описание проекта

Терминальное приложение "Конвертор" — это учебно-демонстрационное приложения для создания и редактирования списка задач.

**Цель проекта:**
- Изучение инструментов и технологий для создания приложений
- Изучение паттернов проектирования
- Изучение создания запросов к реляционным базам данных
- Изучение работы сборщика проектов Gradle
- Изучение принципов ООП
- Изучение возможностей ЯП Java
- Создание приложения для создания и редактирования списка задач
